Graphical User Interface with CakePHP¶

Installation¶

Laudatio-Repository uses CakePHP 2.0, a PHP MVC framework.

Model/Controller¶

XMLObject¶

This is the main model of the laudatio-repository website. It represents the teiHeader by providing methods to manage the data stored in Fedora Commons.

User¶

User, Group, LDAP, CorpusCreator Controllers and Models are used to manage access rights to modify, view or delete corpora.

Configuration¶

The Configuration Controller provides methods for the repository administrators. That includes the configuration of the used scheme for uploading corpora, the view configuration, a form to upload new schemes, buttons to reindex all corpora in elasticsearch and to update all handle pids.

Views¶

View¶

The presentation of the teiHeader contents is supported by a plugin named treetable. A JSON file is used to map the teiHeader content to fields in that table. (See Configuration) The plugin was modified to asynchronously load the content of a clicked field from the server.

Search¶

The search view is based on the facetview plugin. See ElasticSearch for more information.

Import¶

The import view provides a form to upload new coprora that match the active scheme. It is possible to add multiple datastreams to that corpus, to chose the creative commons license and visibility settings for other users.

Modify¶

A user that created a corpus can modify it in the modify-view. Users can change the settings specified at import and upload new versions of teiHeaders an datastreams.

Plugins¶

ElasticSearch¶

We use ElasticSearch to index and search teiHeaders of uploaded corpora. The view of the search is based on FacetView , that is a pure javascript frontend. Users can use the full text search and facets to filter the repository for interesting corpora. See ElasticSearch for more information.

Acl¶

We use the access control list system of Cakephp via the Alaxos ACL Plugin. It provides an interface to configure the rights managed through ACL.

LicenseChooser¶

We use the LicenseChooser Widget to select and display the license of the corpora. The license chosen by the user is stored in Fedora Commons.
